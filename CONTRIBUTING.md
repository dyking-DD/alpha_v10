# 参与贡献指南

感谢你对 Alpha V10.6 的关注！欢迎任何形式的贡献。

## 如何贡献

### 报告 Bug
- 在 Issues 中提交，描述复现步骤和系统环境

### 提交代码
1. Fork 本仓库
2. 创建功能分支：`git checkout -b feature/xxx`
3. 提交修改：`git commit -m "描述你的修改"`
4. 推送分支：`git push origin feature/xxx`
5. 发起 Pull Request

### 提交规范
- 代码需通过现有测试
- 新增功能请附带说明文档
- 涉及风控/交易的修改需详细说明逻辑

## 核心贡献方向
- 策略模块扩展
- 风控引擎增强
- 交易所适配（币安、Bybit 等）
- 文档完善与翻译
- 回测框架集成

## 沟通
- 通过 GitHub Issues 讨论技术问题
- 重大改动请先提 Issue 征求反馈

所有贡献者将列入 README 致谢名单。
